## 서론
구글 Plus 에 가입하여 틈틈이 코딩 대회 준비 관련 내용을 찾던 중에, 잘 정리된 내용의 블로그를 찾았다. 내가 찾던 딱 알맞은 제목의 블로그와 정리들이 있어 모든 내용을 하나씩 진행하다 보면 대회에 참가하여 약간의(?) 성과를 보고자 한다. [Competitive Programming: Algorithms And Data Structure](https://stepupanalytics.com/competitive-programming-algorithms-and-data-structure/) 란 제목으로 된 글이며, 기초 수학, 자료구조 그리고 알고리즘 순이 기본이며 하위에 초보자(Beginner), 중급자(Intermediate), 고급(Proficient) 그리고 전문가(expert)로 구분이 되어 두서 없이 진행하던 것을 계획있게 진행할 수 있고, 어느 정도 부분에서 이해가 어려 운지를 파악할 수 있다는 점에서 "자기 평가"가 될 것이다.
